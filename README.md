# Nidhi_Ahya Node Project

## Prerequisite (Development)

| Module | Version |
| --- | --- |
| Node | 18.8.0|
| Npm | 8.18.0 |
| Mongodb compass | 1.35.0|


## Install Dependencies

npm install


## Running Project In Local
``` bash
$ git clone -b <branchName> <gitUrl> <projectName>

$ npm install

```
## Run app
 
$ nodemon index.js

```
